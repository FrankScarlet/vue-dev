# router-v1

## 介绍

对`router`进行了初步的学习，写了一个显示起步的页面（`main.js`）。实践发现如果是在这种`cli`创建的项目中使用`template`写法，需要加一个`vue.config.js`，把`RuntimeCompiler`加进去，因为这个模板要在运行时渲染。
